# LP HTML素材セット

## 画像

- `hero-background.png`：FV用の背景写真。文字・カードを取り除いた状態です。`background-image`として使用し、左側にHTMLのコピーとCTAを重ねてください。
- `icon-region.png`：対応エリアのアイコン。
- `icon-property-types.png`：山林・空き家・原野・土地・別荘地のアイコン。
- `icon-professionals.png`：専門家連携のアイコン。
- `icon-difficult-property.png`：境界不明・残置物のアイコン。
- `source-design-reference.png`：元デザインの参照用（実装には使用しない）。

## HTML化する文言

### FV

- バッジ：関西の相続不動産に特化
- 対象：山林・空家・原野・土地・別荘地に対応
- 見出し：子どもに「負動産」を残したくない
- 補足：あなたの代で安心して手放しませんか？
- ベネフィット：固定資産税・管理・相続の悩みから解放
- CTA：LINEで引き取り可能か相談

### 4つの特徴

1. 関西の相続不動産に特化（大阪・兵庫・京都・奈良・滋賀・和歌山）
2. 山林・空家・原野・土地・別荘地に対応
3. 司法書士・土地家屋調査士とワンストップ対応
4. 境界不明・荷物残ったままでもOK

## 実装メモ

- 背景画像は横幅いっぱいで表示し、`background-position: center`から開始して、画面幅に応じて右寄せへ調整してください。
- タイトル、説明、CTA、4カード内テキストはすべてHTMLテキストにしてください。
- アイコンPNGは白背景付きの切り出しです。カード背景を白にするか、デザイナー側でベクター化すると最もきれいに仕上がります。
