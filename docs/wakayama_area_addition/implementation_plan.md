# 和歌山県（山林・果樹園・空き家）エリア特設ページ追加の実装計画

本計画は、和歌山県エリア（紀美野・有田川等の耕作放棄果樹園、紀南の急傾斜山林、山間部の空き家など）の対応開始に伴い、地元の人が具体的にイメージできる超特化型の特設ページ（`/wakayama/`）を新規構築し、総合トップページと連携する手順をまとめたものです。

## ユーザー確認事項

> [!NOTE]
> * **和歌山県特設ページの作成**: `/wakayama/index.html` を新設し、既存のLP構成・デザインをベースにしつつ、和歌山県の特定地域に強くフォーカスしたコピーに変更します。
> * **地元の具体イメージ（SEO・AIO）の強化**: 
>   * **紀北・紀中エリア（紀美野町・有田川町・日高川町）**: 高齢化で耕作放棄された急傾斜のみかん畑や柿畑などの果樹園跡地（農地）、山間部の古い農家住宅。
>   * **紀南エリア（那智勝浦町・古座川町・串本町・新宮市）**: 台風や大雨での土砂崩れリスクが心配な急傾斜の山林、何十年も放置されたスギ・ヒノキの保安林。
>   * **山間部エリア（田辺市龍神・中辺路、高野町）**: 相続が何代も放置され登記が古いままの境界不明な森林、山深く管理に行けない古民家。
>   * これらを具体的なお悩み文言として盛り込み、AI検索（AIO）や地域名検索（SEO）でのヒット率を最大化します。
> * **総合トップとの連携**: ポータルの対応エリア部分にある「和歌山エリア窓口（準備中）」を正式に「和歌山エリア窓口」に変更し、`/wakayama/` へリンクします。JSON-LDの `areaServed` に「和歌山県」を追加します。

## 提案される変更点

### 1. 和歌山エリア特設ページの構築
* 新しいディレクトリ `wakayama` を作成し、`wakayama/index.html` を新設。
* 静的アセットへのパスは親ディレクトリ（`../styles.css`、`../script.js` 等）に設定。
* 和歌山県の各対象エリアの超ピンポイントな地名やお悩みをHTML本文、およびJSON-LDの `areaServed` に包含。
* お悩み箇所に「耕作放棄されたみかん畑・柿畑の処分」「紀南の急傾斜山林の災害リスク」「山深く境界不明な相続山林」などを明記。

### 2. ポータル（総合トップ）の更新
* `index.html` の対応エリアセクションにある「和歌山エリア窓口」カードを準備中から正式公開（遷移リンク `wakayama/`）へ変更。
* ポータルの JSON-LD 内の `areaServed` に `和歌山県` を追加（※すでに含まれていますが、他項目との整合性を確認）。
