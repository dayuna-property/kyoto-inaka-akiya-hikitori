# 負動産リサイクルセンター Webサイト ビジュアルリニューアル計画

## 概要
既存サイトの**テキスト過多な構成**を、**図解・イラスト・アイコン中心**のビジュアル重視デザインにリニューアルします。
文章を最小限にし、ワンスクロールで直感的にサービス内容が伝わる構成を目指します。

---

## 現状の問題点

| セクション | 現状 | 問題 |
|---|---|---|
| ヒーロー | テキスト多め | 視覚的インパクト不足 |
| お悩み（Problem） | テキスト+小画像 | もっとビジュアルで共感を誘いたい |
| ABOUT US | **長文が大量** | 読まれない可能性が高い |
| なぜ有料引き取り | テキストのみ | 図解がゼロ |
| リサイクルスキーム | SVG図解あるが小さい | もっとインパクトのある図解が必要 |
| 3つの特徴 | テキスト+小画像 | 視覚的に弱い |
| 比較表 | テーブルのみ | ビジュアル比較に変更 |
| 完全自動連携 | HTML図解+テキスト | 生成画像に置換 |
| 3つの安心宣言 | テキストのみ | アイコン図解が欲しい |
| 手続きの流れ | テキストのみ | ステップ図解に変更 |

---

## リニューアル方針

### 📐 基本ルール
1. **文章は各セクション最大2〜3行**（現状の1/3以下に削減）
2. **生成画像の図解を各セクションに配置**
3. CSSアニメーション・ホバー効果で動きのあるデザイン
4. 既存の機能（FAQ、フォーム、ハンバーガーメニュー等）は維持

## 変更対象ファイル

### [MODIFY] [index.html](file:///c:/Users/tatsu/OneDrive/デスクトップ/物件配信/inaka-akiya-hikitori/index.html)

各セクションのテキストを大幅に削減し、図解画像を挿入：

- **ABOUT USセクション**: 4段落の長文 → 1行キャッチ + 図解
- **なぜ有料引き取りセクション**: テキスト4ブロック → 1枚のインフォグラフィック画像
- **リサイクルスキームセクション**: SVG + テキスト → 生成画像のフロー図
- **比較セクション**: HTMLテーブル → ビジュアル比較画像
- **自動連携セクション**: HTML図 + テキスト → 生成画像
- **安心宣言セクション**: テキスト3カード → 図解画像 + 短いテキスト
- **手続きの流れセクション**: テキスト3ステップ → 図解画像

### [MODIFY] [styles.css](file:///c:/Users/tatsu/OneDrive/デスクトップ/物件配信/inaka-akiya-hikitori/styles.css)

## 検証計画

### 手動検証
- ブラウザでindex.htmlを開き、全セクションが正しく表示されるか確認
- 生成画像が正しく表示されるか確認
- レスポンシブ表示（モバイル・タブレット）に問題がないか確認
- 既存機能（FAQ、フォーム、ハンバーガーメニュー）が引き続き動作するか確認
