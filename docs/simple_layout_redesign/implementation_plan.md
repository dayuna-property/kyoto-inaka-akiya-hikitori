# 実装計画：シンプルで訴求力の高いレイアウトへのリデザイン

この計画書は、「株式会社 神領不動産」のLPレイアウトおよび訴求方法を詳細に分析し、現在のグリーン基調の配色（フォレストグリーン）を維持しながら、キーメッセージが明確で直観的な「売れるLP構成」へとサイトを刷新するための設計書です。

---

## 変更の目的と方針

1. **キーメッセージの視覚的強調**
   * ヒーローセクションで、「何についてのサイトか」「どのような悩みを解決するか」が瞬時に伝わる特大の太文字コピーを配置します。
2. **安心感と親しみやすさの付与**
   * 司法書士や土地家屋調査士などの「専門家チームの顔が見える化（ABOUT）」セクションを新設し、プロフィールや誠実な想いを提示します。
3. **お悩み提起とサービスメニューのカード化**
   * 神領不動産の「お悩みイラストカード」や「はみ出し人物画像付き売却メニュー」を参考に、直感的に内容が頭に入るデザインを実装します。
4. **お役立ち/サポート情報のボタン化**
   * FAQや税金・費用の解説など、読みたい情報へ直感的にアクセスできるアイコン付きグリッドボタンを構築します。
5. **マルチデバイス・全エリアでの統一表現**
   * 共通の [styles.css](file:///c:/Users/tatsu/OneDrive/デスクトップ/物件配信/inaka-akiya-hikitori/styles.css) のみを更新し、インラインスタイルを排除した上で、ルートおよび5つの全エリアページのHTML構造を統一します。

## 神領不動産LPの分析と落とし込み設計

### 1. ヒーローセクション（ファーストビュー）
* **神領不動産の特徴**: 特大の「春日井市の 空き家 空き地 お困りではありませんか？」という文字、右側に代表者の親しみやすい顔写真、左下に「売却相談」のスタンプ。
* **落とし込み**:
  * グリーン背景の上に、白と黄色の特大太文字で配置：
    「＼ 関西主要エリアの ／ **売れない山林・空き家 お困りではありませんか？**」
  * 右側には、笑顔で親身に対応する「専門家チーム（司法書士・土地家屋調査士）」のフラットイラスト（新しくAIで生成）を大きく配置します。
  * 左下に「登記移転引き取り相談」のスタンプ風バッジをCSSで装飾して配置。

### 2. 共感（お悩み）セクション
* **神領不動産の特徴**: 手書き風タイトル「相続した実家や土地どうしたら良いの？」、悩みを持つ人物イラストと吹き出しが付いたシンプルな細枠カードが並ぶ。
* **落とし込み**:
  * タイトルを「**相続した実家や土地、どうしたら良いの？**」と丸みを帯びたCSSフォント装飾で大きく提示。
  * 悩みを以下の5つのシンプルな細枠カードにし、お悩みの表情のイラスト（AI生成）を配置：
    1. 「どこに、何を、どう相談したら良いかわからない…」
    2. 「思い出の実家を処分（有料引き取り）するのに、少し罪悪感がある…」
    3. 「名義や境界が複雑で、ご近所に知られたくない…」
    4. 「家の中に家具や荷物が残ったままで、他社に断られた…」
    5. 「税金だけを払い続けていて、手放す決断ができない…」

### 3. 選ばれる3つの特徴 (FEATURE) セクション
* **神領不動産の特徴**: 左上にオレンジ色の大きな丸数字（01, 02, 03）とはみ出した人物写真、カード下部に強烈なベタ塗り色のリンクボタン。
* **落とし込み**:
  * 既存の「選ばれる理由」を、左上にグリーンの丸数字（01, 02, 03）と専門家イラストが配置された「はみ出し型カード」にリデザインします。
  * カード下部はグリーンのベタ塗り背景にし、矢印付きのリンクボタン（「提携司法書士の登記セットアップについて ＞」など）を構成し、信頼性を高めます。

### 4. 解決・ご提案（ABOUT US / 想い）セクション 【新規追加】
* **神領不動産の特徴**: 「不動産屋らしくない代表が、相続の相談を受けています」というコピーと、代表の大きな写真、プロフィール詳細、想いを語る動画。
* **落とし込み**:
  * 司法書士・土地家屋調査士の専属チームが「なぜこの引き取りサービスを立ち上げたのか」の想いを伝えるセクションを新設します。
  * キャッチコピー：「**不動産会社に断られた物件も、法的な安心と共にお引き受けします**」
  * 専門家たちのイラストプロフィールを、神領不動産風のレイアウト（すっきりとした表と小さな紹介写真/イラストの配置）で追加。

### 5. サービスメニュー（引き取りプラン）セクション
* **神領不動産の特徴**: オレンジ背景の上に白文字で「仲介売却」「不動産買取」を並べ、上部に人物写真がはみ出している。
* **落とし込み**:
  * グリーン背景（#1b4332）の上に、白文字で「**“現状有姿で引き取る”“山林・空き家を価値に変える” あなたに最適な引き取りプランをご提案**」。
  * その下に、2つの「はみ出しイラスト付きプランカード」を並べます：
    1. 「**空き家・古民家 引き取り**」（建物・家財残りOKプラン）
    2. 「**山林・原野・雑種地 引き取り**」（境界曖昧・相続未了対応プラン）

### 6. お役立ち・サポートグリッドボタン 【新規追加】
* **神領不動産の特徴**: 家の背景の上に、右側に4つの分かりやすい白角丸のボタン。
* **落とし込み**:
  * 対応エリア・FAQの近くに、不安を解消するためのお役立ちボタン（4つ）をグリッド配置します。
    * 「引き取り処分のポイント ＞」
    * 「かかる費用・税金の仕組み ＞」
    * 「登記移転完了までの流れ ＞」
    * 「よくあるご質問（FAQ） ＞」
