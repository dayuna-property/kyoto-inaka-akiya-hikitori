# 兵庫県（淡路島・山間部）エリア特設ページ追加の実装計画

本計画は、兵庫県エリア（淡路島、但馬の豪雪地帯、丹波・西播磨の山間部など）の対応開始に伴い、地元の人が具体的にイメージできる超特化型の特設ページ（`/hyogo/`）を新規構築し、総合トップページと連携する手順をまとめたものです。

## ユーザー確認事項

> [!NOTE]
> * **兵庫県特設ページの作成**: `/hyogo/index.html` を新設し、既存の滋賀・京都LPの構成・デザインをベースにしつつ、兵庫県の特定地域に強くフォーカスしたコピーに変更します。
> * **地元の具体イメージ（SEO・AIO）の強化**: 
>   * **淡路島エリア（淡路市・洲本市・南あわじ市）**: 釜口や五色などのバブル期リゾート別荘地、沿岸部の塩害対策が必要な空き家・古民家。
>   * **但馬エリア（豊岡市・養父市・朝来市）**: ハチ高原周辺や山間部の豪雪による倒壊寸前の実家、雪下ろしができず放置された建物。
>   * **丹波エリア（丹波篠山市・丹波市）**: 相続が数代にわたり放置された境界不明の山林や、解体費が数百万かかる広大な旧家。
>   * **西播磨エリア（宍粟市・佐用町）**: 昭和の原野商法で買わされた境界不明の急傾斜山林や放置原野。
>   * これらを具体的なお悩み文言として盛り込み、AI検索（AIO）や地域名検索（SEO）でのヒット率を最大化します。
> * **総合トップとの連携**: ポータルの対応エリア部分に「兵庫エリア窓口」のカードを追加し、`/hyogo/` へリンクします。JSON-LDの `areaServed` に「兵庫県」を追加します。

## 提案される変更点

### 1. 兵庫エリア特設ページの構築
* 新しいディレクトリ `hyogo` を作成し、`hyogo/index.html` を新設。
* 静的アセットへのパスは親ディレクトリ（`../styles.css`、`../script.js` 等）に設定。
* 兵庫県の各対象エリアの超ピンポイントな地名やお悩みをHTML本文、およびJSON-LDの `areaServed` に包含。
* お悩み箇所に「淡路島の塩害空き家・放置リゾート別荘」「但馬の豪雪実家倒壊リスク」「丹波・西播磨の境界不明な相続山林」などを明記。

### 2. ポータル（総合トップ）の更新
* `index.html` の対応エリアセクションに「兵庫エリア窓口」カードを追加。
* ボタンリンクを `hyogo/` に接続。
* ポータルの JSON-LD 内 hometown に `兵庫県` を追加。
