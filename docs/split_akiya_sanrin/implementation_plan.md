# 実装計画：空き家引き取り専用サイトと山林引き取り専用サイトの分割

本計画は、現在「空き家」と「山林」の両方を扱っている引き取り専用サイトを、それぞれのニーズに特化した専用のサイト構造へ分割・再編成するためのものです。

## ユーザー確認事項

> [!IMPORTANT]
> 1. **URL構造の変更に伴う影響**:
>    各エリアページ（滋賀、京都など）の階層が1つ深くなり、`shiga/` から `akiya/shiga/` および `sanrin/shiga/` に変更となります。
> 2. **共通のスタイル・スクリプト**:
>    デザインの統一性とメンテナンス性を維持するため、既存 of `styles.css` および `script.js` を共通リソースとして各ページから読み込みます（相対パスの調整を行います）。

## 変更内容

### 1. ゲートウェイTOPページ (`index.html`) のリニューアル
* 既存の総合LPを「空き家・古民家」と「山林・原野・雑種地」のどちらに進むかを明快に選択できるポータルページ（分岐ゲートウェイ）に変更します。
* シームレスでモダンなヒーローセクションを構築し、2つの選択ボタンを目立たせて配置します。

### 2. 空き家引き取り専用サイト (`akiya/`) の構築
* **総合TOP (`akiya/index.html`)** と **都道府県別ページ (`akiya/*/index.html`)** を構築。
* **コンテンツの特化**:
  * SEOメタデータ（タイトル、ディスクリプション、キーワード）を空き家・古民家処分に最適化。
  * ヒーローコピー、お悩みセクション、想定ケース、引き取りプランから山林関連の内容を排除。
  * リサイクルスキーム（空き家再生・古民家活用）にフォーカス。
  * 問い合わせフォームの「物件種別」選択肢を空き家・古民家関連（家財あり/なし等）に変更。
* **パスの調整**:
  * CSS、JS、画像、リンク先のパスを `akiya/` および `akiya/*/` の階層に合わせて正しく調整します。

### 3. 山林引き取り専用サイト (`sanrin/`) の構築
* **総合TOP (`sanrin/index.html`)** と **都道府県別ページ (`sanrin/*/index.html`)** を構築。
* **コンテンツの特化**:
  * SEOメタデータを山林・原野・雑種地処分に最適化。
  * ヒーローコピー、お悩みセクション、想定ケース、引き取りプランから建物・空き家関連の内容を排除。
  * リサイクルスキーム（林業保全・山林再生）にフォーカス.
  * 問い合わせフォームの「物件種別」選択肢を山林・原野関連（保安林含む/農地等）に変更。
* **パスの調整**:
  * 空き家同様、新階層に対応した相対パス（`../../styles.css`等）の調整を行います。

* ローカルでブラウザを起動し、以下の点を確認します。
  1. ルートTOPから「空き家」「山林」それぞれの総合TOPへ正しく遷移できること。
  2. 各総合TOPから都道府県別の専用ページへ正しく遷移できること。
  3. 各ページでCSS、JavaScript、画像が崩れることなく正常に読み込まれていること。
  4. 空き家専用ページ・山林専用ページそれぞれの問い合わせフォームの「物件種別」の選択肢が適切に変更されていること。
  5. フォームの送信処理（JSによる送信デモと成功モーダル表示）がエラーなく動作すること。
