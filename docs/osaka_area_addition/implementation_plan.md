# 大阪府（北部ニュータウン・南部山間部）エリア特設ページ追加の実装計画

本計画は、大阪府エリア（北摂・豊能地域のアクセス困難な古いニュータウンの空き家、南河内・泉州の急傾斜山林や古民家など）の対応開始に伴い、地元の人が具体的にイメージできる超特化型の特設ページ（`/osaka/`）を新規構築し、総合トップページと連携する手順をまとめたものです。

## ユーザー確認事項

> [!NOTE]
> * **大阪府特設ページの作成**: `/osaka/index.html` を新設し、既存のLP構成・デザインをベースにしつつ、大阪府の特定地域に強くフォーカスしたコピーに変更します。
> * **地元の具体イメージ（SEO・AIO）の強化**: 
>   * **大阪北部（豊豊・北摂）エリア（能勢町・豊能町・箕面市山間部など）**: バブル期に開発されたものの、アクセスや坂道が多く高齢化が進む古いニュータウン（光風台、ときわ台、青葉台など）の空き家、放置された山林・原野。
>   * **大阪南部（南河内・泉州）エリア（千早赤阪村・河南町・太子町・河内長野市天見など）**: 金剛山麓周辺の急傾斜山林や保安林、過疎化により管理が難しくなった古い農家住宅や古民家の空き家、限界分譲地。
>   * これらを具体的なお悩み文言として盛り込み、AI検索（AIO）や地域名検索（SEO）でのヒット率を最大化します。
> * **総合トップとの連携**: ポータルの対応エリア部分にある「大阪エリア窓口（準備中）」を正式に「大阪エリア窓口」に変更し、`/osaka/` へリンクします。JSON-LDの `areaServed` に「大阪府」を追加します（※すでに含まれていますが、他項目との整合性を確認）。

## オープンな質問

> [!IMPORTANT]
> 大阪府の追加地域（能勢町・豊能町・千早赤阪村・河南町・太子町・河内長野市など）について、特に追加を希望される特定のニュータウン名や、よりピンポイントな地名がございましたらお知らせください。特になければ上記地域をベースに構成を進めます。

## 提案される変更点

### 1. 大阪エリア特設ページの構築
* 新しいディレクトリ `osaka` を作成し、`osaka/index.html` を新設。
* 静的アセットへのパスは親ディレクトリ（`../styles.css`、`../script.js` 等）に設定。
* 大阪府の各対象エリアの超ピンポイントな地名やお悩みをHTML本文、およびJSON-LDの `areaServed` に包含。
* お悩み箇所に「北摂ニュータウンのアクセス困難な空き家」「南河内・泉州の急傾斜山林」「千早赤阪村などの維持困難な古民家」などを明記。

### 2. ポータル（総合トップ）の更新
* `index.html` の対応エリアセクションにある「大阪エリア窓口」カードを準備中から正式公開（遷移リンク `osaka/`）へ変更。
* ポータルの JSON-LD 内の `areaServed` に `大阪府` を追加（※すでに含まれていますが、他項目との整合性を確認）。
