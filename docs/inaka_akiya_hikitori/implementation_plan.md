# 実装計画書：京都北部 負動産有料引き取りサービスLP制作

京都北部（綾部市、福知山市、南丹市、京丹波町）エリアに特化した、負動産（処分したい空き家・山林・原野など）の「有料引き取りサービス」を紹介する、プレミアムでSEO/AIOに最適化されたランディングページ（LP）を制作します。

---

## ユーザーレビュー要求事項

> [!IMPORTANT]
> 以下の実装方針についてご確認ください。
> 1. **問い合わせ導線**: LINE（外部リンク想定）とメールフォーム（送信処理はデモ用のクライアントサイド処理、または外部連携用）を設置します。
> 2. **AI画像の使用**: 先ほど生成した京都北部の自然と調和するモダンな和風住宅のプレミアムイメージ（`hero.png`）をメインビジュアルとして採用します。
> 3. **AIO（AI検索最適化）**: 構造化データ（JSON-LD）を用いて、AI検索エンジンが「京都北部で負動産を引き取ってくれる会社」として認識しやすいようにします。

### ② コンポーネントおよびファイル詳細

#### [NEW] [index.html](file:///c:/Users/tatsu/OneDrive/デスクトップ/物件配信/inaka-akiya-hikitori/index.html)
*   **SEO/AIO最適化**: 
    *   適切なタイトル、メタディスクリプション、メタキーワード。
    *   `Service`（サービス）、`LocalBusiness`（地域密着型ビジネス）、`FAQPage`（よくある質問）の3つのJSON-LD（構造化データ）を定義。
*   **セマンティックHTML5**: `header`, `main`, `section`, `footer` 等を使用し、見出し構造（`h1`〜`h4`）を正しく設計。
*   **セクション構成**:
    1.  **ヒーローエリア**: キャッチコピー、LINE・フォームCTA、プレミアム背景。
    2.  **お悩み提起**: 負動産を抱えるオーナーのリアルな悩み。
    3.  **背景解説**: 法改正（仲介手数料33万円）と既存不動産会社が扱わない理由。
    4.  **サービスの特徴**: なぜ有料で引き取るのか、メリット（4つの強み）。
    5.  **引き取りの流れ**: 簡単な4ステップ。
    6.  **よくある質問 (FAQ)**: ユーザーの不安解消。
    7.  **お問い合わせ（CTA）**: LINE導線 ＋ ガラスモルフィズム風Webフォーム。

## 検証計画

### 自動・手動検証
1.  **レスポンシブ確認**: 開発者ツールを使用し、スマホ（iPhone SE〜Pro Max相当）からPC（4K含む）までレイアウト崩れがないことを確認します。
2.  **HTML/CSSバリデーション**: セマンティックタグの整合性、リンク切れの有無を確認します。
3.  **JSON-LD構造化データの妥当性**: AI・検索エンジンに正しく読み取られるよう構文チェックを行います。
4.  **フォーム動作確認**: 入力漏れがあった場合の警告、正常送信時のメッセージ表示が動作することを確認します。
