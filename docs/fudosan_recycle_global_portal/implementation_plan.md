# 負動産リサイクルセンター 総合トップ・京都LP訴求強化の実装計画（仕様アップデート版）

本計画は、ユーザーの行動ハードルを下げ、サービスの信頼性とスピード感をより強く伝えるため、他社比較表、プロ集団による自動連携、完全非対面3ステップ、および「相談無料・ワンストップバッジ」等の訴求要素を追加・修正する手順をまとめたものです。

## ユーザー確認事項

> [!NOTE]
> * 総合トップ（`index.html`）と京都エリアLP（`kyoto/index.html`）の双方に、同様のアップデート（比較表、新設セクション、3ステップなど）を適用し、サイト全体の訴求の一貫性を保ちます。
> * デザイン面（比較表や追加バッジ）は `styles.css` に共有スタイルとして定義し、両方のHTMLから呼び出す構成にします。

## 提案される変更点

### 1. 新規追加・修正するセクション

1. **メインビジュアル（MV）サブコピーの修正**
   * 専属の司法書士・土地家屋調査士チームによる「最短1〜2週間での名義切り離し」をアピールする文章に差し替え。
   * 「提携司法書士・土地家屋調査士 連携済」バッジの設置。
2. **【比較表】通常の他社仲介との違い（追加）**
   * 「専門家への相談・手配（個別調整が必要 vs ワンストップ自動連携）」
   * 「手続きにかかる期間（3ヶ月〜1年以上 vs 最短7日〜2週間）」
   の2つの比較項目を持つテーブルを新設。
3. **【新設セクション】プロ集団の「完全自動連携」**
   * 相続登記がまだ完了していない、境界がわからない、といった複雑な案件も専門家チームがダイレクトに調査・段取りを行い、ユーザーは書類郵送だけで待つだけであることを説明。
4. **「完全非対面・郵送のみ」のステップ案内（3ステップへの修正）**
   * STEP 1：スマホで30秒かんたん査定（来店0）
   * STEP 2：プロ集団による裏側での高速セットアップ（対面0）
   * STEP 3：最短7日〜で引き取り完了（鬼早）
5. **「無料相談・ワンストップ」導線の強化**
   * 公式バッジを各所に配置。
   * LINEやフォーム周辺に「専門家チームへの無料相談」をワンタップで促す導線を追加。

## 検証計画

### 手動検証
1. **比較表のレスポンシブ崩れチェック**:
   * モバイル画面で比較表がはみ出さないか（横スクロールやカード化の挙動）。
2. **自動連携セクションの確認**:
   * 総合トップおよび京都LPで、文章が正しくレイアウトされているか。
3. **ステップの確認**:
   * STEP 1〜3にスリム化され、デザインが崩れていないこと。
4. **リンク遷移**:
   * 各ボタンやリンクが機能すること。
