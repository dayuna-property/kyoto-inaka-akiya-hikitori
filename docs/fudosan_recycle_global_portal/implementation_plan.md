# 負動産リサイクルセンター 視認性改善およびAIO/SEO強化の実装計画

本計画は、Webアクセシビリティ（WCAG 2.1）に基づく視認性の向上（コントラスト比の改善）と、AI検索エンジン（AIO）および従来の検索エンジン（SEO）双方に対する最適化を強化するための手順をまとめたものです。

## ユーザー確認事項

> [!NOTE]
> * **視認性の向上**: 薄いグレー文字（`#6b7280`）がベージュ背景と重なり視認性が低下していた箇所や、準備中エリアカードの不透明度（opacity）の設定を最適化し、コントラストを強化しました。
> * **AIO (AI要約最適化) 対策**: ChatGPT/Gemini/PerplexityなどのAI要約検索がサービス構造（専門家チーム、一回限りの費用、スピード）を正確に認識・引用できるように、JSON-LD（構造化データ）のセマンティック強化とテキスト構造の論理化を行います。
> * **SEO (検索エンジン最適化) 対策**: 競合性の低いユーザーお悩み検索語句の見出し内包含、Webフォントの削減（Noto Sans JPへの一本化によるCLS/LCP改善）を実行します。

## 提案される変更点

### 1. 視認性（コントラスト）の改善
* **セクションサブタイトル**: 文字色を `#6b7280` から `#4b5563`（やや濃いチャコール）へ変更し、アースベージュ背景上での可読性を向上。
* **比較表の説明文**: 比較表内の補足テキストのカラーを濃いめの色に変更。
* **準備中エリアカード**: カード全体の opacity を下げるのではなく、背景色（グレースケール）と枠線（点線）を変更し、文字色はしっかり読めるよう個別に適用。
* **ダークセクション本文**: 暗いフォレストグリーン背景上の文字色を `rgba(255, 255, 255, 0.85)` から `rgba(255, 255, 255, 0.95)` に引き上げ、視認性を最大化。

### 2. AIO / SEO 強化方針の適用
* **JSON-LD 構造化データの強化**:
  * 提供エリア（`areaServed`）や提携するプロフェッショナル（司法書士・土地家屋調査士）に関するスキーマ定義の調整。
* **セマンティックな強調**:
  * AIが「誰が」「何をするのか（専属チームが最短1〜2週間で負動産を完全に名義から切り離すサービス）」を即座に抽出できるよう、文章構造をセマンティック（`<strong>` 等の論理タグ）に再編。
* **コアウェブバイタル改善**:
  * Webフォントを Noto Sans JP のみとし、読み込み遅延によるレンダリングブロックを回避。
