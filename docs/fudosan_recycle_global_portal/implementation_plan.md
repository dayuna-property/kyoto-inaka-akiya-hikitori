# 負動産リサイクルセンター 滋賀エリア特設ページ追加の実装計画

本計画は、滋賀県（湖西・北部）エリアの対応開始に伴い、地元の人が具体的にイメージできる超特化型の特設ページ（`/shiga/`）を新規構築し、総合トップページと連携する手順をまとめたものです。

## ユーザー確認事項

> [!NOTE]
> * **滋賀特設ページの作成**: `/shiga/index.html` を新設し、京都LPの構成・デザインをベースにしつつ、滋賀の特定地域（大津市湖西：和邇・蓬莱・比良など、高島市：マキノ・今津など、長浜市：余呉・西浅井など）に強くフォーカスしたコピーに変更します。
> * **地元の具体イメージ（SEO・AIO）の強化**: 比良山麓の放置別荘地や、マキノ・余呉などの湖北・湖西特有の豪雪地帯における空き家・山林維持の「具体的なお悩み」を文言に盛り込み、AI検索（AIO）や地域名検索（SEO）でのヒット率を最大化します。
> * **総合トップとの連携**: ポータルの対応エリア部分に「滋賀エリア窓口」のカードを追加し、`/shiga/` へリンクします。

## 提案される変更点

### 1. 滋賀エリア特設ページの構築
* 新しいディレクトリ `shiga` を作成し、`shiga/index.html` を新設。
* 静的アセットへのパスは親ディレクトリ（`../styles.css` 等）に設定。
* **大津市湖西、高島市、長浜市湖北**の3市におけるピンポイントな地名（和邇、小野、志賀、蓬莱、比良、北小松、マキノ、今津、安曇川、余呉、西浅井）をHTML本文、およびJSON-LDの `areaServed` に完全包含。
* お悩み箇所に「比良山麓の放置別荘地の管理費」「マキノや余呉の豪雪実家の倒壊リスク」「境界不明の湖北山林」などを明記。

### 2. ポータル（総合トップ）の更新
* `index.html` の対応エリアセクションに「滋賀エリア窓口」カードを追加。
* ボタンリンクを `shiga/` に接続。

## 検証計画

### 手動検証
1. **パス・リンクの遷移確認**:
   * ポータル（`/`）から滋賀ページ（`/shiga/`）へのボタン遷移、および滋賀ページのヘッダーからポータルへの戻りリンクの機能確認。
2. **アセットの読み込み**:
   * `/shiga/` にてCSS/JS/共通画像（hero.png）が正しく読み込まれ、デザインシステム（Noto Sans JP統一）が正常に動作しているかの確認。
3. **AIO/SEO メタタグおよび構造化データの確認**:
   * 滋賀の超ピンポイント地名が JSON-LD および meta タグに適切に含まれているか。
