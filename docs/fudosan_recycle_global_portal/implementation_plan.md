# 負動産リサイクルセンター フォント統一・体制図解追加の実装計画

本計画は、ユーザーの行動ハードルをさらに下げ、公式感を高めるための「チーム体制図解（ダイアグラム）」の追加、およびサイト全体のフォントを「Noto Sans JP」へ完全統一するための手順をまとめたものです。

## ユーザー確認事項

> [!NOTE]
> * 見出し等に指定されていた明朝体フォント（Shippori Mincho）を廃止し、サイト全体のフォントを「Noto Sans JP」に統一します。
> * 自動連携セクション（`#alliance`）の内部に、HTMLとCSSで構築されたレスポンシブかつ微細なアニメーション付きの「チーム体制図」を新設します。
> * 総合トップおよび京都LPの双方に同一のアップデートを適用します。

## 提案される変更点

### 1. タイポグラフィの統一
* `styles.css` の `--font-heading` 変数を、`'Noto Sans JP', sans-serif` に書き換えます。これにより、見出しやロゴを含めた全テキストが Noto Sans JP で統一されます。

### 2. チーム体制のビジュアル図解（ダイアグラム）の実装
* 自動連携セクション（`#alliance`）内に、以下のステップで連携するインタラクティブな図解（ダイアグラム）を配置します。
  1. **お客様（ご相談者）**（スマホで30秒査定、対面0・待つだけ）
  2. ─── 矢印コネクタ（ワンストップ窓口） ───
  3. **負動産リサイクルセンター**（窓口一元化・裏側でチームへ自動配給）
  4. ─── 矢印コネクタ（完全自動連携） ───
  5. **専属専門家チーム（提携司法書士・提携土地家屋調査士）**（自動登記セットアップ、相続・未登記案件の解決）

## 検証計画

### 手動検証
1. **フォントの統一確認**:
   * 見出しやロゴを含め、明朝体から「Noto Sans JP」へ正常に切り替わっていることを確認。
2. **チーム体制図解のレスポンシブ動作**:
   * PCでは横並びの美しいステップフロー、モバイル（幅768px以下）では縦並びかつ矢印方向が下向きに切り替わるレスポンシブ対応を確認。
   * コネクタの矢印部分のアニメーションが自然にループしているかを確認。
