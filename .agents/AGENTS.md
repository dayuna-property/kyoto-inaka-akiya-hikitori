# プロジェクトルール

## AI生成記法の残存防止ルール
Web公開用のHTMLファイルなどを生成・編集する際は、AIによる生成であることを想起させる記法やテキストが一切残らないように以下の項目を徹底してください。

1. **Markdown表記の混入禁止**
   * 強調を示す `**` や `*` をHTMLファイル内にテキストとして埋め込まないでください。太字にする場合は必ず `<strong>` や `<b>` タグを使用するか、CSSでスタイリングしてください。
   * リンクを示す `[テキスト](URL)` や見出しを示す `#` をHTMLファイル内にテキストとして残さないでください。
2. **AI特有のメタテキストの排除**
   * 記事やコンテンツの本文中に「以下に記事を作成します」「以上が内容になります」といった、システムプロンプトへの応答を示す前置きや後書きを絶対に含めないでください。
3. **品質チェックの義務化**
   * ファイルを生成または編集した後は、必ず `**` などのMarkdown記号がファイル内に残留していないか検索・検証を行ってください。
